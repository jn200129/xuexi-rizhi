课程概述

模拟电子技术（简称「模电」）研究的是连续变化的电信号的产生、放大、运算与变换。 它是从「电路分析」走向「数字电路」「电力电子」「通信电子」的必经桥梁，也是公认的「电类四大名补」之一。 本笔记按上海交通大学课程主线，把 45 讲内容重组为十大主题，配自绘 SVG 示意图，便于建立完整的知识脉络。

这门课到底在解决什么问题？

一句话：把微弱的电信号不失真地放大，并对其进行运算和变换。麦克风拾取的声音是几毫伏的微弱电压， 扬声器需要的是几瓦的功率；传感器输出的是微小电流，单片机需要的是 0~3.3 V 的标准电压。 模电就是设计中间这一段「放大 + 处理」的电路，核心是三件事：器件（二极管、三极管、运放）→ 单元电路（放大、反馈、运算）→ 系统（信号发生、功率输出）。

一、半导体与 PN 结

1.1 本征半导体

硅（Si）、锗（Ge）是最常用的半导体材料，导电能力介于导体与绝缘体之间。纯净的半导体称为 本征半导体，其原子按共价键规则排列。在常温下，热运动会使少量共价键断裂，产生一对 自由电子与空穴（电子-空穴对）。空穴带正电，可被邻近电子填补，从而像正电荷一样移动。

两种载流子
自由电子：带负电，是导体导电的唯一载流子，也是半导体载流子之一。
空穴：半导体特有，带正电；它的「移动」本质是邻近价电子的接力填补。
温度的影响
温度升高 → 电子-空穴对增多 → 导电能力增强。这正是半导体器件受温度影响大的根源。

1.2 杂质半导体：N 型与 P 型

掺入微量杂质可以大幅改变导电性，并决定多数载流子的类型，这是制造一切半导体器件的基础。

关键：无论 N 型还是 P 型，整体仍然电中性——杂质提供的多子被其原子核固定电荷平衡。 多子浓度由掺杂决定（基本不随温度变），少子浓度由温度决定（对温度极敏感）。

1.3 PN 结的形成与单向导电性

把 P 型和 N 型半导体制作在一起，交界面附近会发生多子扩散：N 区电子向 P 区扩散、P 区空穴向 N 区扩散， 复合后在交界处留下不能移动的离子，形成空间电荷区（耗尽层 / 阻挡层），并建立一个由 N 指向 P 的内电场。 内电场会阻止多子继续扩散、同时推动少子漂移，最终达到动态平衡。

正向偏置（P 接正、N 接负）
外电场削弱内电场 → 耗尽层变窄 → 多子大量扩散 → 导通，电流大。硅管导通压降约 0.7 V，锗管约 0.2 V。
反向偏置（P 接负、N 接正）
外电场加强内电场 → 耗尽层变宽 → 仅少子形成微弱反向饱和电流，近似截止。

结论——单向导电性：PN 结正向导通、反向截止，这是二极管一切应用的根基。

1.4 PN 结的电容效应

PN 结上存储的电荷会随外加电压变化，表现出电容效应，这是限制器件高频性能的根源：

势垒电容 Cb
耗尽层像平行板电容的介质：反向电压增大 → 耗尽层变宽 → Cb 减小。反偏时起主导，且随电压可调——变容二极管（2.4 节）正是利用这一点做电调谐。
扩散电容 Cd
正向导通时，扩散到对方区域的非平衡少子形成电荷积累，其数量随正向电压变化。正偏时起主导，且随电流增大而增大。

结电容 Cj = Cb + Cd

为什么重要：高频时结电容的容抗变小，把信号「旁路」掉，单向导电性变差——这正是第六章三极管混合 Π 模型中 Cπ、Cμ 限制放大电路上限频率 fH 的物理来源；开关电路中它也决定了二极管的反向恢复时间。

二、二极管及其基本电路

2.1 伏安特性与方程

把 PN 结封装并引出两极，就是二极管（P 端为阳极 A，N 端为阴极 K）。其电流-电压关系为：

iD = IS (euD/UT − 1)

其中 UT 为温度的电压当量（常温约 26 mV）。曲线分四段：死区、正向导通区、反向截止区、反向击穿区。

反向击穿的两种机理：雪崩击穿与齐纳击穿

反向电压增大到击穿电压 U(BR) 时，反向电流急剧增大，称为反向击穿。 按物理机理分为两种，二者都属于电击穿——只要外电路限制了电流、不超过额定功耗，撤去电压后 PN 结即可恢复，是可逆的； 若电流过大导致结温过高把 PN 结烧毁，则是不可逆的热击穿。

二极管的温度特性

二极管对温度敏感，本质是少子浓度随温度指数增长（见 1.1/1.2）。工程上记住三条：

  • 正向压降：温度每升高 1 ℃，正向导通压降减小约 2~2.5 mV（负温度系数）——即同一电流下曲线左移；这一特性也被反过来用作温度传感器。
  • 反向饱和电流 IS：温度每升高 10 ℃约增大一倍，故高温下「反向截止」会明显漏电（硅管远好于锗管）。
  • 击穿电压：随温度的变化取决于击穿机理——雪崩击穿正温度系数、齐纳击穿负温度系数（见上表）。

实际计算中，把二极管这条非线性曲线替换成简单的「直线段电路」，由简到繁分三档：

建模思想：所谓「等效模型」，就是用分段直线去近似那条指数伏安曲线——直线段对应的电路只含理想开关、电压源、电阻三种元件，于是非线性问题变成了线性电路计算。 近似取得越细（理想 → 恒压降 → 折线），结果越准但计算越繁；选哪个模型，取决于信号幅度与 0.7 V 相比是否可忽略。 前三种是「大信号（直流）模型」，回答「通不通、通了压降多少」；第四种微变模型只对叠加在 Q 点上的交流小信号负责，回答「曲线在 Q 点附近的斜率是多少」。

微变等效模型（第 5 讲重点）

当二极管上除直流偏置外还叠加了一个幅度很小的交流信号时，工作点只在静态工作点 Q 附近微小摆动， 这一小段曲线可近似为直线——对这个交流小信号而言，二极管就等效为一个电阻，称为动态电阻（微变电阻）rd， 其值是伏安特性曲线在 Q 点处切线斜率的倒数。对二极管方程求导可得：

rd = UT / IDQ ≈ 26 mV / IDQ(mA)  （常温）

要点：① rd 不是常数，随 Q 点电流增大而减小（1 mA 时约 26 Ω，2.6 mA 时约 10 Ω）； ② 分析套路是「先直流求 Q 点 → 再把二极管换成 rd 算交流」——这正是第五章三极管微变等效电路法的雏形， 三极管的 rbe 公式（5.4 节）中的 26 mV/IE 一项就来源于发射结的这个动态电阻。

2.3 典型应用

整流
利用单向导电把交流变脉动直流，半波整流（1 管）、桥式全波整流（4 管）是开关电源、充电器的入口电路。整流后通常接大电容滤波（电容充放电填平脉动，输出约为 1.2 U2），再接稳压环节（稳压管电路见 2.4，或三端稳压器/串联型稳压——本课程未展开，属「直流稳压电源」章节）。
限幅 / 钳位
把信号幅度限制在某范围内，或把波形某一电平「钳」到固定值，保护后级电路。
检波 / 开关
高频信号取包络（检波），或在数字电路中作为单向通路。

2.4 稳压二极管与特殊二极管

稳压管（齐纳二极管）工作在反向击穿区：只要不超过额定功率，反向击穿（电击穿）是可逆的， 且击穿后电流大幅变化时端电压几乎不变——伏安曲线在击穿区非常陡直，利用这一点实现稳压。 使用时必须串联限流电阻，否则会因电流失控发展为热击穿而损坏。

稳压管的主要参数

稳定电压 UZ
规定电流下的击穿电压，即输出的「稳压值」。同型号存在分散性，但对每一只管子是确定的。
稳定电流 IZ 与最大稳定电流 IZM
IZ 是正常稳压的最小工作电流（低于它曲线不够陡、稳压效果差）；IZM 由额定功耗决定。工作电流必须落在 IZ ~ IZM 之间。
动态电阻 rZ = ΔUZ/ΔIZ
击穿区曲线越陡 rZ 越小（几 Ω ~ 几十 Ω），稳压性能越好。
额定功耗 PZM = UZ·IZM
超过则由电击穿转为热击穿而烧毁。
温度系数 α
由击穿机理决定（见表 2-1）：UZ<4 V（齐纳击穿）为负、UZ>7 V（雪崩击穿）为正、5~6 V 附近最小，故要求温度稳定的基准常选 6 V 左右的稳压管，或用正温度系数稳压管与正向二极管（−2 mV/℃）串联补偿。

典型稳压电路

输入直流电压 UI 经限流电阻 R 接稳压管（反接：阴极接高电位），负载 RL 与稳压管并联取电压。 稳压过程是一个自动调节：UI 升高（或负载电流减小）→ UO 略升 → 稳压管电流急剧增大 → R 上压降增大 → 把多余电压「吃掉」，UO≈UZ 基本不变。 设计时校验两个边界：UI 最高且负载最轻时不超 IZM；UI 最低且负载最重时不低于 IZ。

  • 发光二极管 LED：正向导通时发光，压降约 1.8~3 V，需串限流电阻。
  • 光电二极管：反偏，受光照产生光电流，用于光检测。
  • 变容二极管：利用反偏时结电容随电压变化，用于调谐与压控振荡。

三、双极型三极管（BJT）

三极管是模电的「核心放大器件」，用小电流控制大电流，实现电流放大，进而实现电压、功率放大。

3.1 结构与电流分配

BJT 由两个 PN 结、三个区构成：发射区(E)、基区(B)、集电区(C)，分 NPN、PNP 两类。 实现放大的内部条件：发射区掺杂浓度高、基区很薄且掺杂低、集电结面积大。 外部条件：发射结正偏、集电结反偏。

电流放大系数：

直流：β̄ = IC / IB    交流：β = ΔIC / ΔIB

3.3 特性曲线与主要参数

输入特性 iB~uBE（固定 UCE）：形状与二极管正向特性相同（发射结就是一个正偏 PN 结）， 硅管死区约 0.5 V、导通后 UBE≈0.7 V。UCE 从 0 增大时曲线右移； UCE>1 V 后集电结已充分反偏、基区复合电流基本定型，曲线不再明显移动——所以通常只画 UCE≥1 V 一条。

输出特性 iC~uCE（固定 IB）：一族曲线，分三个区（见图 4）。

主要参数
电流放大系数 β（几十~几百）、极间反向电流 ICBO/ICEO（受温度影响大）。二者关系：ICEO = (1+β)·ICBO——基极开路时集-射间的「穿透电流」是 ICBO 被放大 (1+β) 倍的结果，故 β 大的管子温度稳定性反而差。
极限参数：集电极最大电流 ICM、最大反向电压 U(BR)CEO、最大功耗 PCM，三者围成「安全工作区」。
基区宽度调制（Early 效应）
放大区的输出特性并非绝对水平，而是微微上翘：UCE 增大 → 集电结耗尽层变宽 → 有效基区变薄 → 复合减少、β 略增 → IC 略升。把各条曲线向左反向延长会近似交于一点（−UA，Early 电压）。上翘斜率的倒数就是集-射间的动态电阻 rce（几十~几百 kΩ）——它是 H 参数中 h22、混合 Π 模型中 rce 的物理来源，也决定了 7.3 节「有源负载」能把增益提到多高的上限。
温度的影响
温度升高 → β 增大、ICBO 增大、UBE 减小（约 −2 mV/℃）→ 静态工作点漂移，这是放大电路必须「稳定 Q 点」的原因。

四、场效应管（FET）

场效应管是电压控制器件——用栅源电压 uGS 控制漏极电流，几乎不取栅极电流， 因此输入电阻极高（可达 109 Ω 以上）。它是 MOS 集成电路、CPU、功率开关（MOSFET）的基础。

4.1 分类

  • 结型场效应管 JFET：靠反偏 PN 结的耗尽层宽度控制沟道。
  • 绝缘栅型 MOSFET：栅极用 SiO₂ 绝缘，分增强型（需 UGS 才形成沟道）与耗尽型（零偏即有沟道），又各分 N 沟道、P 沟道。

4.2 N 沟道增强型 MOSFET（最常用）

当 uGS 超过开启电压 UGS(th) 时，栅极正电压在衬底感应出 N 型反型层（导电沟道），漏极才有电流。 其工作区分为：可变电阻区（深三极管区）、恒流区（放大用）、夹断/截止区。

4.3 特性曲线与主要参数（第 8~9 讲）

FET 用两组曲线描述（对应 BJT 的输入/输出特性）：

转移特性 iD~uGS（固定 uDS）
体现「电压控电流」。恒流区内近似为平方律： 增强型 MOS：iD = K(uGS − UGS(th))²； JFET / 耗尽型：iD = IDSS(1 − uGS/UGS(off))²， 其中 IDSS 为 uGS=0 时的饱和漏极电流，UGS(off) 为夹断电压。
输出特性 iD~uDS（固定 uGS）
一族曲线，分可变电阻区（uDS 小，iD 随 uDS 近似线性，可作压控电阻）、恒流区（放大用，对应 BJT 放大区）、截止区与击穿区。

主要参数：

  • 开启电压 UGS(th)（增强型）/ 夹断电压 UGS(off)（耗尽型、JFET）：形成/夹断沟道的临界栅源电压。
  • 低频跨导 gm = ΔiD/ΔuGS（单位 mS）：转移特性在 Q 点的斜率，衡量栅压对漏流的控制力，是 FET 小信号模型的核心参数（地位相当于 BJT 的 β），恒流区内 gm 随 IDQ 增大而增大。
  • 直流输入电阻 RGS：JFET >107 Ω，MOS 管可达 109~1015 Ω。

FET 的小信号（微变）等效模型

与二极管、三极管一样，FET 在恒流区的 Q 点附近也可线性化。因为栅极几乎不取电流（iG≈0）， 输入回路 g-s 之间直接视为开路；输出回路 d-s 之间等效为一个压控电流源 gm·ugs（受栅源电压控制，gm 即转移特性在 Q 点的斜率）， 必要时再并联输出特性上翘对应的电阻 rds（通常几十 kΩ~几 MΩ，估算时忽略）。

使用注意：MOS 管栅极与沟道间隔着极薄的 SiO₂，输入电阻又极高，感应的静电电荷无处泄放，极易静电击穿——存放时应使三个电极短接，焊接时烙铁要良好接地，切勿悬空栅极。

FET 的偏置电路（设 Q 点）

FET 放大同样要先偏置在恒流区，但栅极不取电流，偏置只需「造一个合适的 UGS」： 自给偏压式——源极串电阻 Rs，ID 流过使源极电位升高，栅极经大电阻接地（0 V），得 UGS=−IDRs， 只适用于 UGS<0 也能导通的 JFET / 耗尽型； 分压式——栅极由 Rg1/Rg2 分压抬到正电位，再配 Rs，UGS=UG−IDRs 可正可负，增强型必须用它。 求 Q 点时把偏置方程与转移特性（平方律）联立即可；Rs 同样起稳定 Q 点的直流负反馈作用（对应 BJT 的 5.5 节）。

5.4 动态分析：微变等效电路法（H 参数模型）

命名说明：这里把三极管线性化得到的小信号模型，正式名称叫 H 参数（混合参数）等效模型（第 13 讲）——以 rbe（即 hie）和电流放大系数 β（即 hfe）为核心参数，故又称「rbe-β 模型」。

在小信号下，把三极管在 Q 点附近线性化为一个受控源模型：基极-发射极间等效为 rbe，集电极等效为受控电流源 βib。

「H 参数」怎么来的：把三极管当成一个二端口网络（输入口 b-e、输出口 c-e），在 Q 点附近对两条特性曲线做全微分，各写一个线性方程：

ube = h11·ib + h12·uce     ic = h21·ib + h22·uce

四个 h 参数就是特性曲线在 Q 点的四个偏导数（量纲有电阻、有纯数、有电导，「混合」hybrid 由此得名）。其中 h12（输出电压对输入的内反馈，约 10-3~10-4）和 h22（输出特性微小上翘的斜率，即 1/rce，rce 高达几十~几百 kΩ）在一般精度下都可忽略， 于是只剩两个参数：h11 = rbe（输入特性在 Q 点的切线斜率倒数）和 h21 = β——模型简化为「一个电阻 + 一个受控电流源」：

rbe 的组成也有清晰的物理来源：基极到发射极要先过基区的体电阻 rbb'（约 100~300 Ω），再过发射结——发射结就是一个正偏 PN 结，其动态电阻 re=26 mV/IE（即 2.2 节二极管的 rd！）； 但 re 上流的是 ie=(1+β)ib，按 5.6 节的阻抗折算规则，折算到基极回路要乘 (1+β)：

rbe = rbb' + (1+β)·26 mV / IE(mA) ≈ 200 Ω + (1+β) · 26 mV / IE(mA)

模型的适用边界（易错点）：① 只对交流增量有效——先画交流通路（直流电源 VCC 对地短路、耦合/旁路电容短路）再替换三极管； ② 信号要小（ube 摆幅不超过约 10 mV，否则指数曲线的弯曲不可忽略，产生非线性失真）； ③ 只适用于中低频（高频时结电容不可忽略，要换 6.1 的混合 Π 模型）； ④ 管子必须已由直流偏置在放大区——所以永远是「先静态，后动态」。

由此求得共射放大电路的核心指标：

电压增益 Au = −β(Rc∥RL) / rbe

负号表示共射电路输出与输入反相（180°）。这是共射组态的标志性特征。

输入电阻 Ri = Rb ∥ rbe     输出电阻 Ro ≈ Rc

5.5 静态工作点的稳定——分压式偏置电路

温度变化会使 Q 点漂移。分压式偏置（射极偏置）电路用 Rb1、Rb2 分压固定基极电位 UB， 再用射极电阻 Re 引入直流负反馈：温度↑→IC↑→UE↑→UBE↓→IB↓→抑制 IC 上升，从而稳住 Q 点。 Re 并联旁路电容 Ce，保证交流增益不受损。

把两种组态级联，可取长补短，是高性能放大与集成电路里的常见结构（第 17 讲）：

六、放大电路的频率响应

放大倍数并非对所有频率都相同。由于电容的存在，增益在高、低频两端都会下降。把增益（dB）与相位随频率（对数）变化画出来，就是波特图。

低频段下降的原因
耦合电容、旁路电容的容抗随频率降低而增大，分压使增益下降，决定下限频率 fL。
高频段下降的原因
三极管极间电容（结电容）、密勒效应使高频信号被旁路，决定上限频率 fH。
通频带
BW = fH − fL，即增益下降到中频值 0.707 倍（−3 dB）之间的频率范围。

估算方法：每个电容与它「看进去」的回路等效电阻 R 构成一阶 RC 环节，转折频率都是同一个公式：

f = 1 / (2πRC)

  • 耦合/旁路电容与回路电阻构成高通 → 各自给出一个下限频率，取最高者为 fL（常由容量小、回路电阻小的旁路电容 Ce 决定）；
  • 结电容/杂散电容与等效电阻构成低通 → 各自给出一个上限频率，取最低者为 fH。

在转折频率处增益恰好下降 3 dB，相位产生 ±45° 附加相移；远离转折频率后每十倍频变化 ±20 dB、相移趋向 ±90°。

6.1 高频分析工具：混合 Π 模型

研究高频特性要用混合 Π（hybrid-π）模型（第 24 讲）替代低频的 H 参数模型：在 b'-e 间加入结电容 Cπ、b'-c 间加入结电容 Cμ。 其中 Cμ 跨接在输入与输出之间，因密勒效应被等效放大 (1+|Au|) 倍折算到输入端，成为限制高频的主因。

密勒效应的定量结论：共射电路中 Cμ 两端电压是 ub'e 与被放大反相后的输出之差，充放电电流被放大，从输入端看等效于一个大得多的对地电容。折算后总输入电容与上限频率为：

Ci ≈ Cπ + (1 + gmRL′)·Cμ     fH ≈ 1 / (2π·R·Ci)（R 为 Ci 两端看到的等效电阻）

这解释了 5.7 节 Cascode 电路展宽频带的原理：共基级把共射管的电压增益钳到约 1，使 (1+gmRL′) 因子消失，Cμ 不再被放大。

β 的频率响应
电流放大系数 β 本身随频率下降，定义 β 截止频率 fβ（β 降到 0.707 倍处）。
特征频率 fT
β 下降到 1（0 dB）时的频率，是三极管高频性能的核心指标，近似有 fT ≈ β0·fβ。

把增益幅值（dB）与相位随频率（对数横轴）的关系用折线近似画出，就是波特图（第 24 讲）：每个极点使幅频以 −20 dB/十倍频下降、相位滚降 90°。多级电路的波特图由各级叠加。

增益带宽积近似恒定：放大倍数越高，带宽往往越窄。多级放大时，总下限频率升高、总上限频率降低，总带宽变窄。

七、差分放大与集成运算放大器

7.1 差分放大电路——直接耦合的基石

集成电路内部无法做大电容，多级间采用直接耦合，由此带来零点漂移问题。 差分放大电路用两个对称的管子，对差模信号（两输入之差）放大、对共模信号（两输入相同部分，含漂移、干扰）抑制，巧妙解决了零漂。

共模抑制比 KCMR = | Ad / Ac | （越大越好）

射极接恒流源可大幅提高对共模信号的抑制能力，是集成运放输入级的标准结构。

7.3 电流源电路与有源负载（镜像电流源）

集成电路里电阻又大又不精确、还占面积，因此偏置和负载大量改用电流源电路（第 27~29 讲）。 最基本的是镜像电流源：两个特性一致的管子，基极相连、一管接成二极管形式作「参考」， 另一管便「复制」出与参考电流近似相等的输出电流（IC2 ≈ IREF），故名「镜像」。

作恒流偏置
给差分对管尾部提供稳定的恒定电流，大幅提高共模抑制比 KCMR。
作有源负载
用电流源代替集电极电阻 Rc，因其动态电阻极大，单级电压增益可提高几个数量级——这是集成运放中间级实现高增益的关键。

常见改进型：比例电流源、微电流源、威尔逊（Wilson）电流源、多路电流源，目的都是提高复制精度与输出电阻。

7.4 理想运放与「虚短、虚断」

分析含运放电路时，做两条理想化假设（开环增益 Aod→∞、输入电阻→∞、输出电阻→0）：

虚短（线性区，有负反馈时）
因增益无穷大而输出有限，故两输入端电压几乎相等：u+ ≈ u-。若同相端接地，则反相端为「虚地」。
虚断
因输入电阻无穷大，两输入端电流几乎为零：i+ = i- ≈ 0。
这两条是分析一切运放运算电路的「万能钥匙」——绝大多数运算电路的公式都由「虚短+虚断」两步推出。注意：虚短只在负反馈、工作于线性区时成立；开环或正反馈（如比较器）时不成立。

八、负反馈放大电路

把输出量的一部分送回输入端，与输入反向叠加，就是负反馈。它以牺牲增益为代价，换取放大器各项性能的全面改善，是几乎所有实用放大器的核心。

闭环增益 Af = A / (1 + AF)   其中 (1+AF) 称为反馈深度

当 AF >> 1（深度负反馈）时，Af ≈ 1/F，增益几乎只由反馈网络（无源元件）决定，与器件参数无关——这正是运算电路精度高、稳定的根本。

8.1 四种反馈组态

按输出端取样方式（电压/电流）和输入端比较方式（串联/并联）组合成四类：

组态取样比较稳定量对电阻影响

判别口诀：取样——反馈取自输出端节点是「电压」，取自输出回路电流是「电流」（看反馈线是否与负载并联）； 比较——反馈信号与输入信号接在同一节点是「并联」，接在不同节点是「串联」。正负反馈用瞬时极性法判断。
例题：用瞬时极性法判断正/负反馈（四步法）

方法：① 假设输入信号某瞬时为「+」（电位升高）；② 沿放大电路逐级推各点的瞬时极性（共射/反相端 → 反相记「−」，共集/同相端 → 同相记「+」）；③ 推出反馈网络送回输入端的反馈信号极性；④ 看反馈信号是削弱净输入 → 负反馈；增强 → 正反馈。

示例（同相比例运放，Rf 从输出接回反相端）：

  1. 设同相端输入瞬时为「+」。
  2. 运放同相放大 → 输出端 uo 也为「+」。
  3. uo 经 Rf、R1 分压，把「+」送回反相端。
  4. 净输入 = u+ − u−，反相端电位被抬高使净输入减小 → 负反馈。

再判组态：反馈取自输出电压节点 → 电压取样；反馈信号（反相端）与输入信号（同相端）接在不同节点 → 串联比较。故为电压串联负反馈，对应稳定输出电压、提高 Ri、降低 Ro。

8.2 负反馈对性能的改善

  • 提高增益稳定性（增益变化被压缩 1+AF 倍）；
  • 展宽通频带、减小非线性失真与噪声（同样被压缩 1+AF 倍）；
  • 按组态改变输入/输出电阻（串联升 Ri、电压降 Ro……）。
代价：各项改善都以增益下降为交换——被压缩的倍数与改善的倍数同为 (1+AF)。「性能不会凭空变好，只是用多余的增益去换」。

8.3 负反馈放大电路的稳定性（第 35 讲）

负反馈电路自带隐患：放大电路在高/低频段有附加相移（每个极点最多贡献 90°，见第六章）。 若在某个频率上附加相移达到 −180°，反馈信号就从「反相抵消」变成「同相增强」——负反馈变成了正反馈； 此时若环路增益幅值仍 |AF| ≥ 1，电路便无输入也能输出，发生自激振荡（这两个条件恰是第十章振荡器故意满足的条件）。

判稳方法（环路增益 AF 的波特图）
找两个特征频率：|AF|=1（0 dB）处的 fc，和附加相移 = −180° 处的 f180。 fc < f180 则稳定（幅值先掉到 1，相位还没转够）；fc > f180 则自激。
相位裕度 γ
γ = 180° − |∠AF(fc)|。工程上要求 γ ≥ 45°，留有余量才能保证接上各种负载、温度变化后仍不振、且阶跃响应不过分振铃。
消振：相位补偿
在电路内加补偿电容（简单滞后补偿、RC 滞后补偿、密勒补偿等），人为制造一个低频主极点，让 |AF| 提前跌到 0 dB——用带宽换稳定。多数通用运放（如 741）内部已做「内补偿」，任意闭环增益下都稳定。
经验：三级及以上放大器深度负反馈最容易自激（三个极点能凑满 180°）；反馈越深（F 越大）越危险。若示波器上看到电路「无输入却有高频正弦输出」，先怀疑自激，检查补偿与布线寄生电容。

九、运算电路（运放的线性应用）

9.1 基本运算电路

在运放外接电阻、电容构成深度负反馈，利用「虚短+虚断」即可实现各种数学运算。下表的公式都可由两步推出，建议自己动手验证。

9.2 仪表放大器（高精度差分放大）

单运放差分电路输入电阻低、且要求电阻严格匹配。仪表放大器（第 39 讲）用三个运放改进： 前级两个同相放大器分别缓冲两路输入（输入电阻极高、不衰减传感器信号），后级一个差分放大器做相减。 只需调一个增益电阻即可整体调增益，共模抑制比高、增益可调、输入阻抗大，是测量、传感器信号调理的标准电路。

9.3 模拟乘法器及其应用

模拟乘法器（第 38 讲）输出与两路输入之积成正比 uo=K·uX·uY，是一种重要的非线性运算单元。配合运放可实现：

  • 乘方 / 开方：两输入接同一信号得平方；置于反馈回路得开方。
  • 除法：把乘法器放进负反馈环实现相除。
  • 调制 / 解调、混频：通信中把信号搬移到不同频段。
  • 有效值（RMS）检测、功率运算：电压乘电流得瞬时功率。

十、信号发生与功率放大

10.1 正弦波振荡电路

振荡器是「不加输入也能自己产生周期信号」的电路，本质是带正反馈的放大器。维持等幅正弦振荡需同时满足：

幅值条件
|AF| = 1（起振时需略大于 1，靠非线性自动稳幅）。
相位条件
∠AF = 2nπ，即反馈必须是正反馈（总相移为 0 或 360°的整数倍）。

组成上需有：放大电路、正反馈网络、选频网络（决定频率）、稳幅环节。常见类型：

  • RC 桥式（文氏桥）振荡器：用 RC 串并联选频，适合低频（几 Hz~几百 kHz），波形好。
  • LC 振荡器（变压器反馈、电感三点式、电容三点式）：适合高频。三点式的组成法则（相位条件的电路化）：与发射极（源极）相连的两个电抗同性质，第三边（接基极-集电极）为异性质——口诀「射同基（集）反」。电感三点式易起振但波形较差；电容三点式波形好、频率上限高（改进型：克拉泼、西勒电路，进一步提高频率稳定度）。
  • 石英晶体振荡器：频率极稳定，用于时钟、通信。

RC 桥式（文氏桥）定量结果（第 40 讲）：RC 串并联网络在 f0 处反馈系数最大为 F=1/3 且相移为 0，故放大部分用同相比例电路、增益略大于 3 即可起振：

振荡频率 f0 = 1/(2πRC)     起振条件 Au = 1 + Rf/R1 > 3 （即 Rf 略大于 2R1）

稳幅靠反馈支路的非线性元件自动完成：如 R1 用正温度系数热敏电阻/钨丝灯泡，或 Rf 并联二极管——幅度增大时增益自动回落到 3，输出稳定为低失真正弦波。

10.2 电压比较器（第 42~43 讲）

运放开环或正反馈时工作于非线性区（虚短不再成立，虚断仍成立），输出只有正、负饱和值（±UOM）两种状态—— 比较器就是「模拟量 → 开关量」的一位判决电路。分析方法固定：求出使 u+=u− 的输入电压即阈值 UT，再画输出对输入的电压传输特性（阶梯状折线）。

10.4 波形产生与信号转换电路

把比较器、积分器、运算电路组合起来，可产生各种波形并在不同物理量间转换（第 44~45 讲）。

方波 → 三角波 → 正弦波
迟滞比较器产生方波，送入积分器得三角波，再经折线/差分对网络整形可逼近正弦波——这是函数发生器的基本链路。
锯齿波发生电路
让积分电容的充电与放电时间常数不同（一快一慢），即得上升缓、下降陡的锯齿波，用于扫描、PWM 调制的载波。
电压-电流 / 电流-电压转换（V-I、I-V）
利用「虚短虚断」让流过负载的电流只由输入电压与一个电阻决定，与负载无关（恒流源）；反之把电流转成电压（如光电二极管取信号）。
电压-频率转换（V-F）
把输入电压转换成频率与之成正比的脉冲，用于模数转换、远距离抗干扰传输。
呼应全局：信号转换电路把前面所有单元（放大、反馈、运算、比较、积分、振荡）综合起来，是模电从「器件 → 单元 → 系统」的收口，也直接通向后续的「数模/模数转换」与数字电路。

学习路线小结

  1. 打地基：理解 PN 结单向导电 → 二极管 → 三极管/场效应管的伏安特性与三个工作区。器件是后续一切的物理基础。
  2. 练核心：吃透共射放大电路的「先静态定 Q、再动态求增益」两步法，掌握微变等效电路。这是全课程的计算主线。
  3. 上台阶：差分放大 → 集成运放 → 牢记「虚短虚断」，从此分立器件思维升级为系统思维。
  4. 串全局：负反馈贯穿始终（稳定 Q 点、改善性能、构成运算电路），振荡是正反馈，功放是大信号——它们都是放大与反馈思想的延伸。
一句话总览：模电就是围绕「放大」展开——用器件实现放大，用反馈驯服放大，用运放标准化放大，最后让放大去做运算、产生信号、输出功率。
