# LyraT-Mini 桌面音箱外壳 · 3D 打印指南

横版桌面式外壳，主体（U 型）+ 顶盖两片，4 颗 M2×10 螺丝固定。

## 尺寸与规格

| 项 | 值 |
|---|---|
| 外形 | 100 × 80 × 40 mm |
| 壁厚 | 2.4 mm（侧面）/ 2.0 mm（底面）/ 2.5 mm（顶盖） |
| 主体材料用量 | 46.6 cm³（PLA 约 58 g 实体，20% 填充约 12 g） |
| 顶盖材料用量 | 19.9 cm³（PLA 约 25 g 实体，20% 填充约 5 g） |
| 屏窗口 | 30 × 34 mm（1.9" IPS ST7789 留 3 mm 边框余量） |
| 喇叭孔 | Ø36 mm（配 Ø40 mm 全音域喇叭） |
| 麦克风阵列 | 前面板 5 × Ø1.2 mm 孔（居屏正上方） |
| 6 键（右侧） | Ø3.5 mm，间距 12 mm，一排 |
| SD 卡口（左侧） | 20 × 3 mm |
| Micro-USB（左侧） | 8 × 4 mm |
| PCB 支撑柱 | 4 根 Ø5，高 12 mm，M2 自攻孔 |
| 顶盖螺柱 | 4 根 Ø5.5，全高 35.2 mm，M2 自攻孔 |

## 打印参数（推荐）

| 参数 | 主体 | 顶盖 |
|---|---|---|
| 材料 | PLA / PLA+ / PETG | PLA / PLA+ |
| 喷嘴 | 0.4 mm | 0.4 mm |
| 层高 | 0.2 mm | 0.16 mm（前面板细节多） |
| 填充 | 20% Gyroid | 15% |
| 打印朝向 | 底面朝下（U 型开口面朝上） | 顶面朝下（沉头孔埋在下方） |
| 支撑 | **有**（前面板屏窗口内表面 + 喇叭圆孔） | 无 |
| 支撑类型 | 树形支撑（tree），仅接触外壁 | — |
| 顶/底层数 | 4 层 | 3 层 |
| 墙数 | 3 层 | 3 层 |
| 打印速度 | 60 mm/s（外墙 30） | 60 mm/s |
| 预计时间 | 约 4h（Bambu X1C） | 约 1.5h |

## 组装

1. 主体打印完 → 清除支撑（前面板屏窗口内可能有支撑残留，用美工刀修边）；
2. 焊装/固定 LyraT-Mini 板到 4 根 PCB 支撑柱（**M2×5 mm 自攻螺丝**）；
3. 接屏排线（SPI 6P）、喇叭线、麦克风延长线、电池 XH2.54；
4. 装屏到前面板屏窗口内（用 3M VHB 泡棉双面胶粘在四边，或热熔胶点固）；
5. 装喇叭到前面板喇叭孔背面（同上，泡棉胶）；
6. 电池贴双面胶固定在底板前段；
7. 检查所有线材没有压到 PCB 边缘；
8. 盖顶盖 → 4 颗 **M2 × 10 mm 沉头螺丝**（十字或内六角均可）拧入 4 根螺柱。

## 二次修改

改任何尺寸都在脚本顶部参数区（`case_generator.py` 前 50 行），如：

- 换更小的屏 → 改 `SCR_W, SCR_H`
- 换更大的喇叭 → 改 `SPK_R` 和 `SPK_CX / SPK_CZ`
- 想在后面板加通风孔 → 复用 `MIC_ARR_*` 那段代码，把 Y 从 `-0.1` 改成 `D-WALL-0.1`
- 想加挂墙孔 → 主体后面板加两个 Φ4 孔

改完命令行重跑：
```cmd
D:\FreeCAD\FreeCAD_1.1.1-Windows-x86_64-py311\bin\freecadcmd.exe case_generator.py
```

约 1 分钟出新 STL。

## 已知折衷（后续可迭代）

- **喇叭腔无独立密封**：低音会漏气，效果一般。可加内部隔板做前后分腔（预留 25%）；
- **屏靠双面胶固定**：不如 4 卡爪牢，但改设计需要精确知道模块 PCB 尺寸；
- **无散热孔**：ESP32 + Wi-Fi + 屏 稳定工作时约 1.5 W，PLA 壳 40℃ 环境下不烫手。若在阳光下暴晒需要加通风孔；
- **底面无脚垫**：建议贴 4 颗硅胶脚垫，防滑防划桌面；
- **无 LED 出光孔**：若加 WS2812 灯带，需要在顶盖或后面板加导光孔。
