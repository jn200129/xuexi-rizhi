## 关联硬件

- **主控**：ESP32-WROVER-E（4 MB Flash + 8 MB PSRAM）
- **播放通路**：ES8311 单声道 Codec → 3.5 mm 耳机 / 4Ω 3W 喇叭
- **录音通路**：板载单麦克风 → ES7243 音频 ADC
- **其他**：MicroSD 卡槽、6 个功能键、AP5056 锂电池充电管理

## 开发环境（本机）

- **位置**：`D:\esp\esp-adf`（GitHub 官方源，v2.7）
- **工具链**：`D:\esp\espressif`（ESP-IDF v5.3.1 + xtensa-esp-elf-gcc 13.2.0）
- **激活**：`set IDF_TOOLS_PATH=D:\esp\espressif && D:\esp\esp-adf\export.bat`
