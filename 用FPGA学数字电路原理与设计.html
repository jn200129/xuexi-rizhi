课程概述

《用 FPGA 来学习数字电路原理与设计》由硬禾学堂出品，西南科技大学信息工程学院 罗良老师主讲，面向电类低年级学生、电子爱好者与硬件工程师。课程把传统「数字电路」教材 与现代 FPGA 工程实践结合：理论讲完一个知识点，就用 STEP-MAX10-02「小脚丫」开发板 配合 Quartus 软件做实验验证，真正做到「学完即能动手」。本笔记将 31 讲内容重组为九大主题，配自绘 SVG 示意图。

这门课到底在解决什么问题？

一句话：用「0 和 1」来表示、运算、存储和变换信息，并用硬件把逻辑功能真正实现出来。 数字电路是计算机、通信、嵌入式、单片机/FPGA 一切「数字世界」的根基。它研究的不是连续信号（那是模电）， 而是离散的高/低电平（逻辑 1/0），核心是三件事： 信息表示（数制码制）→ 逻辑实现（组合/时序电路）→ 工程载体（门器件、存储器、PLD/FPGA）。

一、数字系统中的信息表示

1.1 模拟量 vs 数字量

自然界的物理量（温度、声音、光强）大多是连续变化的模拟量；而数字系统只处理 离散的数字量——用两个稳定状态（高电平/低电平、有/无、通/断）来表示逻辑 1 和 0。数字信号抗干扰强、易存储、可精确复制、便于用逻辑器件处理， 这正是「数字化」席卷一切的原因。

1.2 数制（进位计数制）

同一个数量，可以用不同的「基数（Radix）」来表示。数字电路最常用四种：

表 1  常用数制对照（0~15）
十进制 D二进制 B八进制 O十六进制 H
0 / 1 / 2 / 30000 / 0001 / 0010 / 00110 / 1 / 2 / 30 / 1 / 2 / 3
4 / 5 / 6 / 70100 / 0101 / 0110 / 01114 / 5 / 6 / 74 / 5 / 6 / 7
8 / 9 / 10 / 111000 / 1001 / 1010 / 101110 / 11 / 12 / 138 / 9 / A / B
12 / 13 / 14 / 151100 / 1101 / 1110 / 111114 / 15 / 16 / 17C / D / E / F
十进制 → 二进制
整数部分「除 2 取余，逆序排列」；小数部分「乘 2 取整，顺序排列」。
二进制 ↔ 八进制/十六进制
二进制每 3 位对应 1 位八进制、每 4 位对应 1 位十六进制，分组转换即可，无需经过十进制。

1.3 二进制的算术与原码、反码、补码

计算机内部用补码表示有符号数，目的是把减法统一成加法，让加法器一套电路同时完成加减运算。

原码
最高位为符号位（0 正 1 负），其余位为数值绝对值。
反码
正数与原码相同；负数为「符号位不变，数值位逐位取反」。
补码
正数与原码相同；负数为「反码 + 1」。补码的最大优点：A − B = A + (−B 的补码)。

二、布尔代数与逻辑化简

2.1 三种基本逻辑运算

一切复杂逻辑都由与（AND）、或（OR）、非（NOT）三种基本运算组合而成。

由它们派生出常用复合门：与非（NAND）、或非（NOR）、异或（XOR）、同或（XNOR）。 其中与非门和或非门被称为「万能门」——单独用一种即可实现任何逻辑功能。

2.2 布尔代数基本定律

表 3  布尔代数常用公式
名称公式
0-1 律A·0=0，A·1=A，A+0=A，A+1=1
互补律A·Ā=0，A+Ā=1
重叠律A·A=A，A+A=A
分配律A(B+C)=AB+AC
吸收律A+AB=A，A+ĀB=A+B
摩根定律A·B=Ā+B̄， A+B=Ā·B̄
摩根定律是化简之王：它把「与」「或」互相转换，是用「万能门」（与非/或非）实现电路、 以及把表达式变形以减少门数的关键工具。

2.3 逻辑函数的标准形式

任何逻辑函数都可写成两种标准式：最小项之和（标准与或式）和最大项之积（标准或与式）。 最小项 mi 对应真值表中输出为 1 的那一行；把所有为 1 的最小项相「或」，即得标准与或式。

2.4 卡诺图（Karnaugh Map）化简

卡诺图是把真值表按格雷码排列的方格图，使逻辑相邻的最小项在几何上也相邻，从而「圈一圈」就能直观化简。

圈卡诺图三原则：① 每个圈包含 2ⁿ 个相邻的 1；② 圈越大越好（消去的变量越多）； ③ 所有 1 都要被圈到，圈可重叠但每个圈都要有「新」的 1。无关项（×）可按需当 0 或 1 处理，让圈更大。

三、组合逻辑电路

3.1 什么是组合逻辑

组合逻辑电路的输出只取决于当前时刻的输入，与电路过去的状态无关——即「无记忆」。 它由门电路连接而成，不含反馈与存储元件。

分析与设计的两条路

分析（电路 → 功能）
逻辑图 → 写表达式 → 列真值表 → 说明功能。
设计（功能 → 电路）
需求 → 列真值表 → 卡诺图化简 → 画逻辑图（常用与非门实现）。

3.2 加法器

半加器只加两个 1 位数，不考虑低位进位：和 S=A⊕B，进位 C=A·B。 全加器再加上低位进位 Ci，可级联成多位加法器。

3.3 常见组合逻辑模块

表 4  典型组合逻辑功能模块
模块功能典型芯片
编码器把 2ⁿ 个输入信号编成 n 位二进制码（如 8→3）74148（优先编码器）
译码器把 n 位码还原为 2ⁿ 个输出（如 3→8）74138
数据选择器 MUX多路输入选 1 路输出74151（8 选 1）
数据分配器1 路输入分到多路输出74139

竞争与冒险（Hazard）：由于各路信号经过的门级数不同，传输延迟有差异， 输出在切换瞬间可能产生毛刺（尖峰脉冲）。可通过增加冗余项、加滤波电容或用同步时钟采样来消除。

四、时序逻辑电路

4.1 时序 vs 组合：有了「记忆」

时序逻辑电路的输出不仅取决于当前输入，还取决于电路原来的状态——即「有记忆」。 它的核心是存储元件（触发器），并通过反馈把现态送回输入。

4.2 触发器（Flip-Flop）

表 5  四种基本触发器
类型特性方程特点
RSQn+1=S+R̄·Qn（约束 RS=0）最基本，有禁止态
DQn+1=D最常用，「打一拍」延迟，FPGA 寄存器即 D-FF
JKQn+1=J·Q̄n+K̄·Qn功能最全，无约束
TQn+1=T⊕QnT=1 翻转、T=0 保持，用于计数
边沿触发的意义：现代同步设计几乎都用上升沿 D 触发器。所有寄存器在同一个时钟边沿同时更新， 状态切换整齐可控，是 FPGA「同步设计」的基石。

4.3 常见时序逻辑模块

寄存器 / 移位寄存器
多个 D-FF 并联可暂存多位数据；级联并让数据逐位右移/左移，即移位寄存器，可实现串↔并转换。
计数器
对时钟脉冲计数。按进位方式分异步（行波）与同步；按模分二进制、十进制（BCD）、任意进制。同步计数器各触发器共用时钟，速度快、无累积延迟。
序列检测器 / 状态机
用有限状态机（FSM）检测特定输入序列。分 Mealy 型（输出与现态+输入有关）和 Moore 型（输出只与现态有关）。

4.4 同步时序逻辑的设计步骤

  1. 由需求建立状态图 / 状态表；
  2. 状态化简、状态编码（常用二进制或独热码 one-hot）；
  3. 选触发器，求驱动方程与输出方程（用卡诺图化简）；
  4. 画逻辑图，检查能否自启动（无效状态能回到有效循环）。

4.5 异步时序逻辑电路

异步电路没有统一时钟，状态变化由输入直接驱动，速度可以很快，但容易出现竞争-冒险和不稳定， 分析时需借助激励表、状态流图仔细判断。工程上能用同步就尽量用同步——这也是 FPGA 设计的金科玉律。

五、数字门电路与器件

5.1 从「逻辑」到「电路」

前面把逻辑当作抽象的 0/1，本章关心它们如何用真实的晶体管实现。两大主流工艺： TTL（双极型）和 CMOS（场效应型）。

表 6  TTL 与 CMOS 对比
特性TTLCMOS
基本器件双极型三极管 BJTMOS 场效应管（P+N 互补）
静态功耗较大极小（几乎只在翻转时耗电）
电源电压5 V 固定3~15 V，范围宽
抗干扰 / 扇出一般抗干扰强、扇出大、输入阻抗高
速度快早期慢，现代已很快
现状逐渐淘汰主流（FPGA 内部即 CMOS）

5.2 CMOS 反相器原理

CMOS 反相器由一个 PMOS 和一个 NMOS 串联组成：输入为低时 PMOS 导通、NMOS 截止，输出被拉高； 输入为高时反之，输出被拉低。无论稳态如何，总有一管截止，故静态电流近乎为零，这是 CMOS 低功耗的根本原因。

5.3 几个重要概念

OC / OD 门（集电极/漏极开路）
输出可线与、可接不同电平、可驱动较大负载，需外接上拉电阻。
三态门（Tri-State）
除 0、1 外还有高阻态 Z，是构成总线（Bus）的关键——多个器件分时共用一条数据线。
逻辑电平与噪声容限
VIH/VIL（输入门限）、VOH/VOL（输出电平），二者之差即噪声容限，越大抗干扰越强。

六、存储器与可编程逻辑器件

6.1 半导体存储器

表 7  常见存储器分类
类别是否掉电保存典型代表
ROM（只读）是（非易失）掩膜 ROM、PROM、EPROM、EEPROM、Flash
RAM（随机）否（易失）SRAM（快、贵）、DRAM（密、需刷新）

存储器通过地址译码器选中存储单元，容量 = 2地址线数 × 数据位宽。 ROM 本质上也可看作一个固化的真值表，可用来实现任意组合逻辑函数。

6.2 可编程逻辑器件（PLD）

从简单到复杂，PLD 的发展是本课程「用 FPGA 学数电」的落脚点：

PROM / PLA / PAL / GAL
早期 PLD，基于「与阵列 + 或阵列」可编程，规模小。
CPLD
复杂可编程逻辑器件，基于乘积项，掉电不丢失，适合中小规模逻辑。
FPGA（现场可编程门阵列）
基于查找表（LUT）+ 触发器 + 可编程互连，规模大、灵活，可实现极复杂的数字系统。本课程使用的 小脚丫 STEP-MAX10 即 Intel/Altera MAX10 系列 FPGA。

七、数模与模数转换

真实世界是模拟的，数字系统是离散的，二者之间靠 DAC（数→模）和 ADC（模→数）架桥。

7.1 DAC（数模转换器）

把数字量按权转换成对应的模拟电压/电流。最经典的是 R-2R 倒 T 形电阻网络，只需两种阻值即可实现高精度。

7.2 ADC（模数转换器）

把连续模拟信号转换成数字码，分四步：采样 → 保持 → 量化 → 编码。常见类型：

表 8  常见 ADC 类型对比
类型原理特点
并行比较型（Flash）多比较器同时比较最快，但耗资源、位数低
逐次逼近型（SAR）二分法逐位逼近速度/精度/成本均衡，最常用
双积分型对输入、基准两次积分精度高、抗干扰强、慢，用于数字电压表
Σ-Δ 型过采样 + 噪声整形高分辨率，用于音频/测量
采样定理（奈奎斯特）：采样频率必须大于信号最高频率的 2 倍，才能不失真地恢复原信号，否则产生混叠。

八、脉冲发生与处理（555 定时器）

8.1 脉冲电路三件事

数字系统需要时钟与各种波形。脉冲电路负责产生（多谐振荡器）、整形（施密特触发器）、 定时（单稳态）三类任务。

8.2 施密特触发器

它有两个不同的翻转门限（回差电压），能把缓慢变化、带毛刺的信号整形成边沿陡峭的标准方波， 广泛用于波形整形、消抖、抗噪。

8.3 555 定时器——经典中的经典

555 内部由 2 个比较器、1 个 RS 触发器、1 个放电管和 3 个 5kΩ 电阻（故名 555）组成， 外接少量阻容即可构成三种典型电路：

表 9  555 的三种典型应用
接法功能关键参数

9.3 Verilog 入门示例

用 Verilog 描述一个上升沿 D 触发器，体会「行为级描述」如何对应硬件：

// 上升沿 D 触发器
module dff (
    input  wire clk,    // 时钟
    input  wire rst_n,  // 异步复位（低有效）
    input  wire d,      // 数据输入
    output reg  q       // 数据输出
);
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            q <= 1'b0;  // 复位
        else
            q <= d;     // 时钟上升沿锁存
    end
endmodule

再看一个分频/计数器，是 FPGA 最常用的模块之一：

// 模 N 计数器（产生分频时钟）
module counter #(parameter N = 50_000_000) (
    input  wire clk,
    input  wire rst_n,
    output reg  clk_out
);
    reg [25:0] cnt;
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n) begin
            cnt <= 0; clk_out <= 0;
        end else if (cnt == N/2 - 1) begin
            cnt <= 0; clk_out <= ~clk_out;  // 翻转，得到分频方波
        end else begin
            cnt <= cnt + 1'b1;
        end
    end
endmodule
写 Verilog 的硬件思维：always @(posedge clk) 里用非阻塞赋值 <= 描述时序逻辑（寄存器）；纯组合逻辑用 assign 或 always @(*) 配阻塞赋值 =。永远记住：你是在「描述电路」而不是「写程序」。

学习路线小结

把 31 讲串成一条清晰主线：

  1. 打基础：数制码制 → 布尔代数 → 卡诺图化简，掌握「0/1 世界」的语言与运算；
  2. 建逻辑：组合逻辑（无记忆）→ 时序逻辑（有记忆，触发器/计数器/状态机），这是数电的两大支柱；
  3. 识器件：TTL/CMOS 门电路 → 存储器 → PLD/FPGA，理解逻辑如何落到真实硅片上；
  4. 通接口：DAC/ADC 连接模拟世界，555 提供时钟与脉冲；
  5. 勤实践：每个模块都在小脚丫 FPGA 上用 Quartus + Verilog 实测，做到「学—练—证」闭环。
一句话总结：数字电路 = 用 0/1 表示信息（数制码制）+ 用门搭逻辑（组合/时序）+ 用器件落地（门/存储/FPGA）。 而 FPGA 让我们能「即学即做」——理论不再停留在纸上，而是真正跑在硬件里。
