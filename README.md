## 🌐 在线访问（GitHub Pages）

仓库启用 GitHub Pages 后，访问地址为：

```
https://<你的GitHub用户名>.github.io/<仓库名>/
```

首页为 `index.html`。

## 🗂️ 文件结构

```
.
├── index.html                  # 项目首页（学习日志列表）
├── 万用表使用入门到精通.html      # 第 1 篇：万用表完整笔记
```

## ✨ 特点

- 纯静态、单文件自包含，无任何外部依赖，可离线打开
- 语义化标签：`header / nav / main / article / section / figure / table / dl / aside / footer`
- 无障碍优化：跳转链接、表头 `scope`、SVG `role/aria-label`
- 所有插图为自绘 SVG，无视频字幕、无第三方图片
